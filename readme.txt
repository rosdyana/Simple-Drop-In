== Description ==

This plugin is using API of Dropbox. You can choose your file and put the link of your file in your dropbox account into Wordress posting. 

== Installation ==

1. Upload `simple-drop-ins.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Post or Page edit screens.
4. Login to your dropbox account
5. Choose the file you want to put the link, copy the link into your post or page.
